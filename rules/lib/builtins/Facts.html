Facts

{% dynamic setvar source_file "src/main/java/com/google/devtools/build/lib/bazel/bzlmod/Facts.java" %} {% include "_buttons.html" %} User-provided data attached to a module extension that is persisted across reevaluations of the extension. This type supports dict-like access (e.g. `facts["key"]` and `facts.get("key")`) as well as membership tests (e.g. `"key" in facts`). It does not support iteration or methods like `keys()`, `items()`, or `len()`.
